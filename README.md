##2.内容说明
###分为两个个文件：
1. Variable.java　　　　前端显示的内容在数据库中的字段
2. NetThread.java　　　 提供交互接口

##3.如何使用
将这两个文件放入android的工程下，修改import就可以使用相关接口 
###step： 
1. 调用NetThread构造函数生成json请求
2. 调用BeginDeal()开始发送请求
3. 调用getActivityData()获取数据
